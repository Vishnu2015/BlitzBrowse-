# BlitzBrowse-
This is a web browser that I made which only works on Linux . It uses the Google search engine to help you find things on the internet.

## How to Use
1. Right-click the file "Blitzbrowse.py and blitzbrowse.desktop "and select **Properties**.
2. Under **Permissions**, check **"Allow executing file as program."**
3. Double-click the file to start browsing!
